诊断控制

执行蜂窝 payload 下载、USSD、重启、飞行模式等操作

蜂窝 payload 测试

临时启用蜂窝数据(PDP)，通过模组蜂窝网络下载 payload，完成后自动关闭

USSD 查询（如查余额）

通过 AT+CUSD 发起，可用于保号/查余额；结果依运营商而定

模组重启

软重启：发送 AT+CFUN=1,1 重新注册网络，不切断供电。
硬重启：通过 ESP32 GPIO27 将 ML307 EN 拉低再拉高，用于 AT 卡死或软重启无效。
这里只重启模组；主控重启在「系统维护」。

飞行模式

飞行模式开启后模组射频关闭，无法收发短信

时间同步 (NTP)

强制向 NTP 服务器重新对时（设备联网后本会每小时自动对时一次）。NTP 服务器在「系统设置」里配置。
